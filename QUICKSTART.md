# Быстрый старт

## Установка

```bash
pip install -r requirements.txt
```

## Запуск экспериментов

Для запуска всех экспериментов выполните:

```bash
python -m src.run_experiments
```

Это создаст директорию `results/` с результатами всех экспериментов. Процесс может занять некоторое время, так как выполняется множество экспериментов с репликациями.

## Анализ результатов

После завершения экспериментов выполните:

```bash
python -m src.analysis
```

Это создаст графики в директории `report/figures/` и таблицы статистики.

## Структура результатов

- `results/all_results.pkl` - все результаты в формате pickle
- `results/summary.csv` - сводная таблица результатов
- `results/*.pkl` - отдельные файлы для каждого эксперимента
- `report/figures/*.png` - графики сравнения метрик
- `report/statistics.csv` - статистика по сценариям

## Настройка экспериментов

Для изменения параметров экспериментов отредактируйте файл `src/run_experiments.py`:

- `num_replications` - количество репликаций (по умолчанию 5)
- Список сценариев в `get_all_scenarios()`
- Конфигурации партий в `main()`

## Примечания

- Для ускорения можно уменьшить количество репликаций или размер партий при тестировании
- Результаты сохраняются автоматически, можно прервать и продолжить позже
- Графики строятся на основе сохраненных результатов
