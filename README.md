# Имитационное моделирование поточной линии окончательной сборки авиалайнеров

Курсовая работа / НИРС по имитационному моделированию сборочного авиазавода.

## Описание проекта

Проект реализует дискретно-событийную имитационную модель поточной линии окончательной сборки авиалайнеров на основе библиотеки SimPy. Модель позволяет исследовать влияние различных конфигураций производственной системы (буферы между участками, параллельная обработка, гибкая рабочая сила) на ключевые показатели производительности.

## Структура проекта

```
project_root/
  src/
    __init__.py
    data.py              # Исходные данные по операциям, участкам, типам самолетов
    model.py             # Описание имитационной модели (классы, процессы)
    scenarios.py         # Различные сценарии экспериментов
    run_experiments.py   # Запуск серий экспериментов и сохранение результатов
    analysis.py          # Анализ результатов, построение графиков
  notebooks/
    experiments_overview.ipynb  # При необходимости обзорный ноутбук
  report/
    report.md            # Текст курсовой работы (на русском)
    figures/             # Сохраненные графики
  results/               # Результаты экспериментов
  requirements.txt
  README.md
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск экспериментов

Для запуска всех экспериментов:

```bash
python -m src.run_experiments
```

Это создаст директорию `results/` с результатами всех экспериментов.

## Анализ результатов

Для анализа результатов и построения графиков:

```bash
python -m src.analysis
```

Графики будут сохранены в директории `report/figures/`.

## Сценарии экспериментов

1. **Сценарий 1**: Буферы между участками отсутствуют (ёмкость 0)
2. **Сценарий 2**: Буфер между участками - 1 самолет
3. **Сценарий 3**: Буфер - неограниченный
4. **Сценарий 4**: Каждый участок может обрабатывать 2 самолета одновременно, буферы отсутствуют
5. **Сценарий 5**: Единый пул рабочих (гибкая рабочая сила)

## Параметры модели

- **Количество участков**: 5
- **Механиков на участок**: 8
- **Типы самолетов**: SA101, SA102, SA103
- **Партии**: 75 самолетов (60% SA101, 40% SA102) и 100 самолетов (45% SA101, 30% SA102, 25% SA103)
- **Время цикла**: 4 дня (96 часов) для партии 75, 3.5 дня (84 часа) для партии 100

## Метрики анализа

- Время цикла конвейера (время между последовательными завершениями самолетов)
- Время завершения партии
- Использование рабочей силы (загрузка участков)
- НЗП (незавершенное производство)
- Блокирование и простои каждого рабочего участка
